# ロールエージェントの追加方法

このテンプレートは「必要なロールだけ使う」設計です。
不要なロールは削除し、自分の職種に合った新しいロールを追加できます。

---

## 新しいロールを追加する手順

### 1. ディレクトリを作成する

```
agents/[ロール名]/
└── .clauderules
```

ロール名の例: `hr`（人事）、`finance`（財務）、`marketing`（マーケ）、`legal`（法務）

### 2. .clauderules を書く

以下のテンプレートをコピーして編集してください。

```markdown
# agents/[ロール名]/.clauderules
# あなたは [ロール名] エージェントです。
# 役割: [このロールが担う業務の説明]

## 業務本能

### [業務カテゴリ1]
- [この職種特有のルール・判断基準]
- [よくある落とし穴と対策]

### [業務カテゴリ2]
- ...

### よく使うテンプレート
- `templates/[テンプレート名].md` — [説明]

## 禁止
- [このロールが絶対にやってはいけないこと]
```

### 3. CLAUDE.md のエージェント構成に追記する

`CLAUDE.md` の「エージェント構成 > ロールエージェント」セクションに1行追加します。

```markdown
agents/[ロール名]/   [説明]
```

### 4. 必要であればテンプレートを追加する

`templates/` に新しい文書テンプレートを追加できます。

---

## 不要なロールを削除する手順

1. `agents/[ロール名]/` ディレクトリごと削除する
2. `CLAUDE.md` のエージェント構成から該当行を削除する

---

## .clauderules を書くコツ

| 書くべきこと | 書かなくていいこと |
|---|---|
| この職種特有の判断基準 | 一般的なビジネスマナー |
| よくある失敗パターンと対策 | 当たり前のこと |
| 使うべきテンプレート | 詳細な手順（別ドキュメントへ） |
| 絶対にやってはいけないこと | |

**参考**: 既存の `agents/pdm/.clauderules` や `agents/sales-planning/.clauderules` を見てください。
